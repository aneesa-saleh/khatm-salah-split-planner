# Qur'an Khatm by Salah Planner

## About the App
Plan your Qur'an Khatm around your 5 daily prayers plus Tahajjud, whatever time works for you.

Options:
- Select number of days
- Select number of pages for each salah
- Flexible page range (excellent for hifdh students who want to revise only some sections of Qur'an)

## Tech
- React + Vite
- Tailwind CSS

## How to contribute
- Clone the project
- Create a branch off the `main` branch and checkout your new branch
- Run `npm i` to install packages
- Run `npm run dev` to run the dev server
- Once you've made updates, open a pull request
